# Install for agents

If you just entered this repo:

1. Read `AGENTS.md`.
2. Read `README.md`.
3. Read `configuration/multi-model-orchestration.md`.
4. Read `security/agent-security-hardening.md`.
5. Read `workflows/sub-agent-patterns.md`.
6. Read `workflows/cross-lane-handoffs.md`.
7. Read `workflows/design-contracts.md`.
8. Use `templates/` when you need a starter contract or handoff file.
9. Use `llms.txt` for the short map.
10. Use `llms-full.txt` for bundled context.
11. Rebuild generated docs with `python3 scripts/build-llms.py` after changing any source doc listed in `llms-config.json`.
